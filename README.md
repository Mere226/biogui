# GUI for sEMG acquisition

## Introduction
Modular PySide6 GUI for acquiring and visualizing sEMG data from serial ports.

## Usage
Run the script [`main.py`](https://github.com/pulp-bio/gui-semg-acquisition/blob/main/main.py).
